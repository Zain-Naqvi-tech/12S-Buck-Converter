## Overview
![Alt Text]("C:\Users\zainn\OneDrive\Desktop\Buck Converter PCB 3D Image.png")
This project is a custom DC-DC Step-Down (Buck) Converter designed in Altium Designer. It takes a higher input voltage and efficiently steps it down to a lower, stable output voltage suitable for powering microcontrollers, sensors, or other low-voltage electronics.

## Specifications
| Parameter | Value |
| :--- | :--- |
| **Input Voltage (Vin)** | [36V - 50.4V] |
| **Output Voltage (Vout)** | [5V] |
| **Max Output Current** | [3A] |
| **Switching Frequency** | [500kHz] |
| **Main Controller IC** | [LMR16030SDDAR] |
